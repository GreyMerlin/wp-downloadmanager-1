From v1.0x To v1.30
  1. Deactivate WP-DownloadManager Plugin
  2. Open wp-content/plugins Folder
  3. Put/Overwrite:
    Folder: wp-downloadmanager
  4. Delete this folder if exists:
    Folder: downloadmanager
  5. Activate WP-DownloadManager Plugin
  6. You Need To Re-Generate The Permalink (WP-Admin -> Options -> Permalinks -> Update Permalink Structure)
  7. Refer To Usage For Further Instructions
» Usage Instructions
General Usage
  1. To embed a specific file to be downloaded into a post/page, use [download=2] where 2 is your file id.
  2. If you are using Default Permalinks, the file direct download link will be 'http://yoursite.com/index.php?dl_id=2'. If you are using Nice Permalinks, the file direct download link will be 'http://yoursite.com/download/2/', where yoursite.com is your WordPress URL and 2 is your file id.
  3. The direct download category link will be 'http://yoursite.com/downloads/?dl_cat=3', where yoursite.com is your WordPress URL, downloads is your Downloads Page name and 3 is your download category id.
  4. In order to upload the files straight to the downloads folder, the folder must be first CHMOD to 777. You can specify which folder to be the downloads folder in Download Options.
  5. You can configure the Download Options in 'WP-Admin -> Downloads -> Download Options'
  6. You can configure the Download Templates in 'WP-Admin -> Downloads -> Download Templates'
Downloads Page
  1. Go to 'WP-Admin -> Write -> Write Page'
  2. Type any title you like in the post's title area
  3. Type '[page_downloads]' in the post's content area (without the quotes)
  4. Type 'downloads' in the post's slug area (without the quotes)
  5. Click 'Publish'
Download Stats (With Widgets)
  1. Activate WP-DownloadManager Widget Plugin
  2. Go to 'WP-Admin -> Presentation -> Widgets'
  3. To Display Most Downloaded
  4. Drag the Most Downloaded Widget to your sidebar
  5. You can configure the Most Downloaded Widget by clicking on the configure icon
  6. To Display Newest Downloads
  7. Drag the Newest Downloads Widget to your sidebar
  8. You can configure the Newest Downloads Widget by clicking on the configure icon
  9. Click 'Save changes'
Download Stats (Outside WP Loop)
  • To Display Most Downloaded
  • Use:
    <?php if (function_exists('get_most_downloaded')): ?>
       <?php get_most_downloaded(); ?>
    <?php endif; ?>

    The first value you pass in is the maximum number of files you want to get.

    Default: get_most_downloaded(10);

  • To Display Recent Downloads
  • Use:
    <?php if (function_exists('get_recent_downloads')): ?>
       <?php get_recent_downloads(); ?>
    <?php endif; ?>

    The first value you pass in is the maximum number of files you want to get.

    Default: get_recent_downloads(10);

  • To Display Downloads By Category
  • Use:
    <?php if (function_exists('get_downloads_category')): ?>
       <?php get_downloads_category(1); ?>
    <?php endif; ?>

    The first value you pass in is the category id.
    The second value you pass in is the maximum number of files you want to get.

    Default: get_downloads_category(1, 10);
